Estructura del Proyecto:
El proyecto sigue una arquitectura sencilla de controlador:
  Paciente.java: Clase entidad que encapsula los datos del paciente.
  ControladorPaciente.java: Clase que gestiona la lógica de almacenamiento y listado.
  T1TECNICASDEPROGRAMORIEOBJ.java: Clase principal con el menú interactivo y captación de datos vía Scanner.
