# T1 - Gestión Hospitalaria XYZ 🏥

Proyecto en Java para el curso de **Técnicas de POO**. Soluciona el caos de datos del Hospital XYZ usando objetos y listas.

## ✨ Características
- **POO:** Uso de clases, atributos privados (encapsulamiento) y listas.
- **Validación:** Control de errores con `try-catch` para que el programa no se cierre.
- **Git:** Control de versiones para el avance del proyecto.

## 📂 Archivos
- `Paciente.java`: Datos del paciente.
- `ControladorPaciente.java`: Manejo de la lista (ArrayList).
- `T1TECNICASDEPROGRAMORIEOBJ.java`: Menú principal y Scanner.

## 🚀 Ejecución
Solo clona, abre en tu IDE y corre el archivo principal. ¡Listo!
